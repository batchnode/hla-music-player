# HLA v1.0.0 - Initial Release

## Overview
HLA (Hyper Local Audio) is a high-performance, local-first music player designed for a modern and fluid audio experience. This initial release establishes the core architecture and features, focusing on automated metadata enrichment and a unique, responsive user interface.

## Key Features
- **Bento-Style Library:** A sophisticated staggered grid layout providing an organized and visually dense overview of your music collection.
- **Automated Metadata Sync:** Integration with MusicBrainz and CoverArtArchive to automatically fetch and cache missing album artwork.
- **Synced Lyrics:** Native support for plain-text and synchronized (LRC) lyrics via LRCLIB integration.
- **Advanced Playback Engine:** Built on `just_audio` with support for gapless playback and custom crossfade simulations.
- **Native Equalizer:** Seamless integration with system-level audio effects.
- **Multi-Directional Navigation:** A custom UI framework allowing seamless horizontal and vertical transitions between the library, equalizer, and now-playing views.

## Supported Audio Formats
- **Standard:** MP3, AAC, WAV
- **Lossless:** FLAC, ALAC (Apple Lossless)
- **Other:** OGG, OPUS, M4A, AIFF

## Technical Specifications
- **Framework:** Flutter
- **Target Platform:** Android (Fully Tested), iOS (Configuration Ready)
- **Minimum Android SDK:** 21 (Lollipop)
- **Target Android SDK:** 34 (Android 14)

## Installation
Download the attached `app-release.apk` and install it on your Android device. Ensure that "Install from Unknown Sources" is enabled in your system settings if prompted.
